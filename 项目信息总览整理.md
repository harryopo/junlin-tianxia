## 一、项目定位（聚焦食用菌 + 美育/研学 + AI 赋能教具）

**深圳信息职业技术大学**「智润客乡」实践团，赴**河源市龙川县**，依托**食用菌企业（河源市粤隆升农业科技有限公司）**开展暑期社会实践。

- 聚焦食用菌企业，团队内统筹 **"产教研学开发 + 商管营销宣传" 双线**（两线并行、共享经费）。
- 吸收组员建议稿：把"菌菇知识问答 + 植物书签 DIY + 菌菇 IP 共创 + 美育成果展"作为**美育/智育融合线**纳入方案。
- **车田豆腐（智造）方向：暂缓 / 放弃**；**AI 智能方舱等技术赋能：不纳入**。
- **AI 赋能教具（本轮新增核心产出）**：自研 **菌菇识别小程序（微信小程序）+ 菌菇知识网页小游戏（H5）+ 研学课程包**，三者均为"AI 赋能"，作为可复用、可转化的数智产品（详见第四、五节）。
- 保留强化：菌菇识别小程序（研学教具）、菌菇网站、研学课程包、包装美化、宣传视频、公众号带货、乡村美育。
- 一句话：用 AI + 数智手段，帮食用菌企业做品牌传播、包装升级、研学课程、媒体发声，**冲击省级奖项**。

## 四、实践内容（聚焦食用菌 + 美育/智育融合 + 研学拓展 + AI 赋能教具）

### 主线 A · 产教研学开发（张子涵）—— AI 赋能教具 + 数智产品

**AI 赋能研学教具包（核心产出）**：围绕食用菌设计一套可复用、可落地的"AI 研学教具包"，含三大件，全部"AI 赋能"：

1. **菌菇识别小程序（微信小程序）**：拍照识菌菇 → 返回品种 / 营养 / 吃法 / 科普卡片，内置"菌菇小问答 + 积分闯关"。参考昆明食用菌研究所"菌窝子 App"（拍照识菌微信小程序）、深圳职业技术大学人工智能学院"农资进销存小程序"（微信小程序框架落地乡村产业）的做法，把识别能力做成轻量小程序，可作创业项目长期转化（服务器 / token 先自垫）。
2. **菌菇知识网页小游戏（H5）**：无需安装的网页互动科普游戏，面向中小学生做趣味科普。参考楚雄州融媒体"接菌子挑战"（滑动竹筐接可食用菌加分、毒菌扣分）、"食用菌栽培工"小游戏（拖动蘑菇到木腐菌 / 草腐菌 / 寄生菌分类、关卡地图 + 3D 模型）、云南"安全食用野生菌"答题闯关（化身采菌人答题赢奖励）等成熟玩法，我们做"菌菇图鉴分类 + 安全采菌闯关"类 H5 小游戏。**AI 赋能点**：题目与科普知识点由大模型生成并审核、识别类玩法调用 AI 图像识别、AI 语音 / 动画讲解——做到"边玩边学、玩中学"。
3. **研学课程包（教案 + PPT + 教具）**：以"识别菌菇 → 画菌菇 → 做菌菇"为路径，面向中小学生 / 企业讲解员；**核心是培训企业讲解员**，留下可复用"课程产品"。课程细化设计后续由张子涵组完成（不急）。

- **包装美化**：食用菌 / 水果外包装设计（企业明确需求）。
- **海报**：初稿完成，待优化。

### 主线 B · 商管营销宣传（吴桐组，依纪要）

### 美育/智育融合（吸收组员建议稿，已纳入）
三大特色活动（源自组员建议稿《三日日程》，已对齐主项目）：
1. **菌菇知识卡片问答**：以食用菌为主题，大学生提问、小学生匿名手写**菌菇知识问答卡片**（如"哪种菌菇最有营养""菌菇生长需要什么条件""灵芝为什么算食用菌"），贴合主项目、收集童真科普语录。
2. **自然植物压花书签 DIY**：采集花草、压花、塑封、装饰，亲手制作专属书签（美育手工，用于成果展）。
3. **小学生手绘 + 大学生 AI 优化 菌菇乡土 IP 共创**：产出原创菌菇 IP 形象 + 海报 + 文创周边。

- 覆盖乡村学生：**具体人数以合作小学为准，本方案不另详列**（原草稿"60+"已下调且不细化）。

### 附 · 参考案例（联网搜集，2026-07-14，支撑教具设计）
- **拍照识菌小程序**：昆明食用菌研究所"菌窝子 App"（微信小程序，拍照识菌 + 知识百科 + 辨识比对）；深圳职业技术大学人工智能学院为乡镇开发的"农资进销存小程序"（微信小程序框架，前端+后端数据库，农户手机直用）。
- **菌菇/农业科普小游戏**：楚雄州融媒体"接菌子挑战"（滑动接可食用菌加分、毒菌扣分的安全科普小游戏）；"食用菌栽培工"教育小游戏（拖动蘑菇到木腐菌/草腐菌/寄生菌分类、关卡地图 + 3D 香菇模型交互）；云南"安全食用野生菌"答题闯关（化身采菌人、答题赢奖励、集电子菌卡）。
- **AI+游戏化研学**：南京信息职业技术学院"图形化编程制作互动游戏""看图识人 AI 图像识别"；广东工业大学"编程闯关游戏拆解诗句""大富翁虚拟棋盘解几何题""AI 动画实时生成"；中国农业大学乡村 CEO 项目"石刻寻宝"小学生游戏化研学课程。
- **结论**：菌菇识别（小程序）+ 趣味分类/闯关（网页小游戏）+ AI 图像识别与语音讲解，是已被验证、可复制、契合乡村中小学生认知的"AI 赋能教具"组合，本方案据此落地。
